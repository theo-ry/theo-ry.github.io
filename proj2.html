Yofro

A static, responsive website of a fictional yoghurt company that includes a database.

MORE
The Challenge

To design and develop the frontend and backend of a website for a new, local startup using information architecture within a few weeks.

The Process

I first decided on my idea(a new yoghurt company) then researched on the current dessert websites that were available and took note of their strengths and weaknesses. I then came up with the background, name and logo of the company. After doing so, it was easier to decide on what the target users would want so I came up with some user flows and the sitemap. I then did more research, sketched wireframes, came up with the theme/ style, made mockups, coded the frontend, coded the backend then linked them to create the website.
