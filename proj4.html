Responsive Design

Nu Vision

Mobile and desktop mockup designs of a content heavy website for a fictional school named “Nu Vision Academy”.

MORE
The Challenge

To create a working prototype of a responsive website for a new Polytechnic, Nu Vision Academy using information architecture within a few weeks.

The Process

I started by doing research on websites of schools and took note of their strengths, weaknesses and of how they organised information. Next, I decided on what the target users would want and came up with some user flows and a sitemap. I then did more research, sketched wireframes, came up with the theme/ style, made mockups and linked them. I got people to user test then made changes until I ended up with the final prototype.
